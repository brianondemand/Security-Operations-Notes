## 1. Foundations of Blue Teaming

* Difference between Red, Blue, and Purple Teaming
* Security frameworks (MITRE ATT\&CK Defender, NIST CSF, CIS Controls, ISO 27001 basics)
* Incident Response Lifecycle (Preparation → Detection → Containment → Eradication → Recovery → Lessons Learned)

## 2. System Security

* **OS Hardening**

  * Windows security (Group Policies, PowerShell logging, Sysinternals, Event Viewer)
  * Linux hardening (auditd, fail2ban, log analysis, SSH hardening)
* **Patch Management & Configuration Management**
* **Endpoint Security**: EDR, AV, USB/device control, host firewalls
* **User & Privilege Management**: Least privilege, PAM, credential hygiene

## 3. Network Security

* Network segmentation & VLANs
* Firewalls (stateful vs next-gen, rule writing)
* IDS/IPS (Snort, Suricata, Zeek)
* Network monitoring with Wireshark & tcpdump
* Secure protocols (SSH, TLS, VPNs, IPsec)
* Common attacks vs defenses (DoS/DDoS mitigation, ARP spoofing, DNS poisoning, MITM defense)

## 4. Security Monitoring & Threat Detection

* Log analysis & correlation (Windows Event Logs, Syslog, Web server logs, Firewall logs)
* MITRE ATT\&CK for defense (mapping detections to TTPs)
* Threat hunting basics (indicators of compromise, anomaly detection)
* Use case development: Detecting brute-force, lateral movement, persistence

## 7. Defensive Engineering & Threat Modeling

* Threat modeling (STRIDE, DREAD, PASTA)

---

## 8. Practical Blue Team Labs

Since your students already know red teaming, design **attack vs defense labs**:

* Detect & respond to brute force (use Hydra + Splunk alerts)
* Detect persistence via scheduled tasks/cron jobs
* Analyze malicious network traffic (Wireshark, Zeek)
* Simulate phishing → create detection rules in SIEM
* Ransomware containment → IR playbook execution

---

✅ **Suggested Learning Flow:**

1. Start with **incident response fundamentals + SIEM/logging** (core SOC skills).
2. Move into **system & network hardening**.
3. Then introduce **threat hunting + forensic analysis**.
4. Finally, advance to **SOC workflows, automation, and threat modeling**.
