### 6.1 Endpoint Security Challenges

Endpoints (laptops, servers, mobile devices) are the primary target for attackers due to their number, diversity, and direct interaction with users. Securing them requires visibility beyond traditional antivirus.

### 6.2 EDR (Endpoint Detection and Response)

EDR solutions continuously monitor endpoints to detect and respond to threats like malware and ransomware. They provide capabilities for:
- **Real-time visibility:** Monitoring process creation, network connections, and file system changes.
- **Investigation:** The ability to search across all endpoints for IOCs.
- **Response:** Remotely containing or remediating a compromised endpoint.

### 6.3 Malware Analysis Basics

The process of dissecting malware to understand its functionality, origin, and potential impact. It is often divided into:
- **Basic Static Analysis:** Examining the malware without running it (e.g., checking file hashes, strings).
- **Basic Dynamic Analysis:** Running the malware in a sandboxed environment to observe its behavior (e.g., what files it creates, what network connections it makes).
